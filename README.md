# VidFlow AI Pro

**Complete AI Shorts & Reels Automation Platform**

Merged version of VidFlow AI + AutoShorts AI.

## Features
- Idea Generation & Script Writing
- Video Generation (Gemini + Veo + Imagen)
- Human-in-the-Loop Approval Pipeline
- Auto Posting to YouTube, TikTok, Instagram Reels, Facebook
- Affiliate Program with 5%–10% recurring commission
- OAuth Channel Management
- GDPR Compliant

## Tech Stack
- Frontend: Next.js 15 (App Router)
- Backend: FastAPI
- Database: Supabase
- AI: Google Gemini, Veo, Imagen
